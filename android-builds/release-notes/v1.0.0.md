# Release Notes - v1.0.0

- **Release Date:** 2026-06-15
- **Developer:** Transport Development Team
- **Status:** Initial Stable Release

---

## 🚀 Features Added
- **Authentication**: Secured login and registration for Students, Parents, Drivers, and Administrators.
- **Live Maps Tracking**: Initial Google Maps platform integration inside the student mobile application.
- **Daily Attendance**: NFC/RFID scanning logic for tracking passenger check-ins and check-outs.

## 🔧 Features Improved
- Optimized location packet payload sizes for lower bandwidth consumption on mobile networks.

## 🐛 Bug Fixes
- Fixed initial WebSocket reconnect handler looping on low network signals.

## ⚠️ Known Issues
- Descending sort logic in RFID taps might ignore limits on specific MongoDB database setups.
